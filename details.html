Réseaux & virtualisation

  • Rappel de bases
    • Fonctionnement simplifié d'une requête
    • DNS, fichiers hosts
  • Pratique: Création d'un projet
    • Création d'un nom de domaine (Studio321.ovh)
    • Création d'un hébergement VPS chez OVH
    • Administration via SSH
      • Installation d'un systeme LAMP
      • Configuration de sous-domaines sur le serveur Apache
    • Création des sous domaines
  • Virtualisation
    • Virtualbox + Ubuntu 16.04 server
    • Installation d'Apache + PHP + mysql
    • Configuration de dossiers partagés dans /var/www
    • Approche de solutions alternatives: Vagrant, Puppet, Ansible (survol)
    • Utilisation d'une configuration PuPHPet (Repo de la config)

Révision de HTML

  • Point sur les connaissances en HTML, CSS
    • Point sur les selecteurs CSS simples et complexes
    • Parler des outils tels que SASS/LESS
    • Parler des frameworks CSS Bootstrap, Foundation, Materialize,..., et des starters comme HTML Boilerplate.
    • Principes de responsive design en CSS
  • Exemples et exercices
    • Création d'un portfolio sans framework css
    • Création d'une page d'accueil pour un site marchand imaginaire
    • Création d'un portfolio avec framework css

Versionning

  • Présentation de GIT (démo de push/pull/reset, historique, ect)
    • Exercices en local
  • Inscription sur Github
  • Création d'une organisation Github
  • Exercices sur une repo de test, forkée par les étudiants:
    • commit / push / pull request
